# 이력서

## 소개

- 이름: 이노원
- 깃허브 저장소: [https://github.com/quavious](https://github.com/quavious)
- 블로그: [https://nwlee.com](https://nwlee.com)

TypeScript와 Next.js를 좋아하는 사람입니다. 자바스크립트 및
타입스크립트 생태계를 좋아하고 유틸리티 타입과 같이
타입스크립트에서 제공하는 기능을 프론트엔드 프로젝트에 활용할 수
있는 방법을 찾고 있습니다. 자바스크립트 트렌드를 놓치지 않기 위해
깃허브 저장소 및 여러 블로그에서 프로그래밍 패턴 또는 새로운
라이브러리를 탐색합니다. 반응형 디자인을 지키기 위해 크롬 개발자
도구에서 화면 너비를 조정해가며 컴포넌트에 대한 UI UX에 문제가
없는지 검토합니다.

## 현재 기술 스택

- 언어: JavaScript, TypeScript
- 라이브러리: React, MobX, SCSS, Styled JSX
- 프레임워크: Next.js, Tailwind CSS
- 개발 도구: Visual Studio Code, Github, Slack, Linear

## 경력 사항

### Metamorp

3D 프린터를 일반인들이 쉽게 조작할 수 있도록 도와주는 웹 기반 애플리케이션입니다.

기술 스택

- TypeScript, Next.js, Tailwind CSS, MobX, Styled JSX

작업 사항

- Next.js, Tailwind CSS로 프론트엔드 개발
- 일렉트론의 IPC 통신으로 프론트엔드에서 필요한 데이터 전송
- 리스트 컴포넌트에서 드래그 앤 드롭으로 각 항목 순서 변경
- 각 컴포넌트에 필요한 인터페이스 개발
- 컴포넌트 외부 영역 클릭했을 때 이벤트 처리
- 컴포넌트 마우스 각 이벤트 핸들러 작성
- 태블릿 PC 화면 사이즈 대응

## 프로젝트 목록

### ARGUMENTS

뉴스 기사에 대해서 유저들이 익명으로 의견을 작성하고 각 의견에 대해 댓글을 게재할 수 있는 웹 어플리케이션입니다.

기술 스택

- TypeScript, Next.js, Prisma, Next-Auth, Tailwind CSS, SCSS, Vercel

작업 사항

- Next.js 점진적 사이트 재생성으로 메인 페이지의 뉴스 기사 업데이트

  - 네이버 뉴스 API

- Next-Auth로 이메일 기반 유저 인증
- Vercel 깃허브 저장소 연동으로 Next.js 어플리케이션 배포 자동화
- Prisma로 데이터베이스 스키마 작성 및 마이그레이션
- react-virtuoso 패키지로 가상 스크롤 컴포넌트

URL

- [깃허브](https://github.com/quavious/arguments-forum)
- [배포 사이트](https://feedarguments.com/)

### 블로그

새롭게 배운 프로그래밍 지식을 저장하기 위한 MDX 기반 블로그

기술 스택

- TypeScript, Next.js, MDX, Tailwind CSS, Styled JSX, Yarn PnP, Vercel

작업 사항

- next-mdx-remote 패키지로 MDX 파일 데이터 Next.js에 렌더링, 특정 태그에 대한 커스텀 컴포넌트 주입
- gray-matter로 MDX 파일 메타데이터 추출
- Yarn Plug n play로 패키지 의존성 관리
- Styled JSX로 컴포넌트 코드 내부에서 스타일 주입

URL

- [깃허브](https://github.com/quavious/guide)
- [배포 사이트](https://nwlee.com)
